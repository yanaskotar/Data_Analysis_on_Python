# BABYNAMES

## Dataset

Набор данных для анализа расположен в `data/babynames/`.

### Data source

- <https://www.ssa.gov/oact/babynames/limits.html>

### Getting the data


```
wget https://www.ssa.gov/oact/babynames/names.zip
mkdir -p data/babynames
unzip names.zip -d data/babynames
```
